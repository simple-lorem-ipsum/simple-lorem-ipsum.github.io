Simple Lorem Ipsum

Simple Lorem Ipsum Webextension for Web Developers

View on GitHub

This extension provides a simple Lorem Ipsum generator. It produces some dummy text for web developers to test form fields.

When using the context menu on a form field, you are able to insert some Lorem Ipsum dummy text. You can also achieve this using the hotkey

  • on Windows: Ctrl+Shift+V
  • on MacOS: Command+Shift+V.

If you want to bind the hotkey to another key, please consider installing this extension.

Requirements

This extension has been successfully tested with:

  • Mozilla Firefox 48.0+
  • Chromium 51.0+
